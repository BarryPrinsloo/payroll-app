Payroll Management Application (Desktop System)

This application is a Windows-based payroll management system designed to help businesses efficiently manage employee payroll processing, salary calculations, and basic HR administration.

Built using Python and PySide6, the system provides a modern graphical user interface with a secure login screen and a central dashboard for navigating core payroll functions.

Key Features

1. Secure Login System
Users must log in with a username and password to access the system. This ensures that only authorised personnel can access payroll data.

2. Main Dashboard
After login, users are taken to a central dashboard where all payroll modules can be accessed. This acts as the control centre of the application.

3. Employee Management (planned expansion)
The system will allow adding, editing, and managing employee records such as:

Personal details
Job role
Salary structure
Banking information

4. Payroll Processing (planned expansion)
The application will calculate employee salaries based on:

Basic salary
Hours worked or overtime
Deductions (tax, UIF, etc.)
Bonuses or allowances

5. Payslip Generation (planned expansion)
Automatically generate payslips for employees that can be viewed, printed, or exported to PDF.

6. Data Storage (planned expansion)
All employee and payroll data will be stored in a local database (such as SQLite), allowing for persistent and structured record keeping.
